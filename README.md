# web

This project is a simple website built using pure HTML and CSS. It reflects my preference for keeping things minimal and straightforward without the use of web frameworks.

## Features

- Lightweight and fast
- Built with clean, structured HTML & CSS
- No external frameworks
- Zero JavaScript.
- Customizable elements (currently `btn` and `link` classes)
